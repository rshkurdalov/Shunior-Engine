# Shunior Engine
This project is CPU rendering, text layout and user interface implementation.<br>
Though it is in development, there are base components ready to use such as 2d graphics output, flow layout and text field UI components.<br>
![Безымянный](https://user-images.githubusercontent.com/17935193/109952501-92f8ca80-7ce7-11eb-8f26-69e80b907674.png)<br>
There are no dependencies for this project, only Windows.h for interaction with operating system.<br>
Also no system float/double types are used. Custom implementation for real numbers provided.
