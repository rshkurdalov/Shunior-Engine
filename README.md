# Shunior Engine
# Features
<ul>
  <li>Rendering 2d graphics with only CPU</li>
  <li>UI components: flow layout, grid layout, button, text field, window</li>
  <li>Following attributes are implemented in base UI frame:
    <ul>
      <li>Position</li>
      <li>Width, height (absolute, relative, autosize, min and max values)</li>
      <li>Paddings, margins</li>
      <li>Visible, focusable, enabled flags</li>
      <li>Event callback functions</li>
    </ul>
  </li>
  <li>Custon real number implementation (no float/double used in library)</li>
  <li>Structures of data: array, set(AVL tree), file based set(AVL tree)</li>
</ul>
